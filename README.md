# AWS CURSO



## Installation


```sh
git clone https://github.com/xronsx/awscourse.git
vagrant up
vagrant ssh
su - awsdev
 - password: spring123
cd springs/
tree -d
```
## Run Project
```sh
cd survey/
python manage.py runserver 0.0.0.0:8000
```
